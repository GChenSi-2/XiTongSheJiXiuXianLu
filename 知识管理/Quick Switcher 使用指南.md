
Quick Switcher 是 Obsidian 自带的核心插件，用来**快速跳转到任意笔记**，是日常使用频率最高的工具之一。

## 一、启动快捷键

| 操作                | 快捷键（Windows）   |
| ----------------- | -------------- |
| 打开 Quick Switcher | **`Ctrl + O`** |
| 关闭                | `Esc`          |
|                   |                |

按下 `Ctrl + O`，屏幕中央会弹出一个搜索框。

## 二、基本用法

### 1. 模糊搜索笔记

直接输入文件名的**任意片段**即可，不需要完整匹配。

例如你想打开 `OSnote/Process_py.md`：

- 输入 `proc` → 列表里就会出现
- 输入 `pypr` → 也能匹配（模糊搜索）
- 输入 `os pr` → 空格分词也支持

### 2. 键盘操作

- `↑` / `↓`：上下选择
- `Enter`：在当前标签页打开
- **`Ctrl + Enter`**：在**新标签页**打开（非常常用）
- `Ctrl + Shift + Enter`：在**新面板**（分屏）打开
- `Shift + Enter`：**创建**一个不存在的新笔记

### 3. 创建新笔记（隐藏技巧）

输入一个**不存在**的文件名，比如 `新笔记测试`，然后按 `Shift + Enter`，就会直接在 vault 根目录创建这个笔记并打开。

## 三、进阶技巧

### 1. 指定文件夹创建

在搜索框里输入 `OSnote/新章节`，按 `Shift + Enter`，会在 `OSnote/` 下创建新笔记。

### 2. 搜索标题 vs 文件名

Quick Switcher 默认只搜**文件名**。如果要搜笔记正文内容，用 `Ctrl + Shift + F`（全局搜索）。

### 3. 最近打开的文件

直接按 `Ctrl + O` 不输入任何字，列表会显示**最近打开**的笔记，方便来回切换。s
